= Rbouncely

A very simple Rails plugin for accessing the Bouncely API.

Bouncely tracks Amazon SES bounces.
<http://bouncely.com>


Install:

copy rbouncely.rb.initializer_example to config/initializers/rbouncely.rb and edit it to contain your API key.


Example:

bouncely = Rbouncely::Bouncely.new(Rbouncely::CONFIG)

bounces = bouncely.get_bounces(Date.today)
